This help page summarizes how to configure Users, Credentials, Groups, Schedules, Doors and Schedule Mappings so door access works as expected. See the README for full details.

  1. Create Users; optionally group them.
  2. Create Credentials (RFID) with correct card-id and nb-bits; optionally set Owner.
  3. Create Schedules with timeframes.
  4. Create Schedule Mappings that connect Schedules to Users/Groups/Credentials and target Doors. If no doors are set, the mapping applies to all doors.
  5. Verify by presenting the credential at the door and checking Audit/ Dashboard for GRANTED/DENIED.

Auth decision is made by Leosac using the mapping and current time. The result is logged as an AuthEvent.
